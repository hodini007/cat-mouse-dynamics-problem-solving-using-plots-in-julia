This repository models the dynamics of a cat chasing a mouse using Julia and visualizes the results with the `Plots` package.

## Getting Started

### Prerequisites

- Julia 1.6+
- Install dependencies:
- 
    ```julia
    using Pkg
    Pkg.instantiate()
    ```

### Usage

Run the simulation script:

```julia
include("your_script.jl")
```
